# Conding on iPad

## Prerequesties
* iPad Pro with iPadOS 13.4^
* USB-C Hub
* Magic trackpad 2
* Bluetooth Keyboard
* HDMI monitor
* Shell emulator
    * iSH
    * Blink Shell

## Methods

### SSH
* hacker mode!
* fun & challenging

### Cloud IDE
* Theia IDE, code-server etc
* easy
* needs expensive VPC

## 쉘 앱

* 현재 아래 두 앱 모두 유니코드 문자열 표시에 문제 있음
* ssh로 맥에 접속해보는데 zsh-autocompletion이 지워지지 않는 이슈로 사용이 힘듬

### ish.app
* x86 에뮬레이션
* SIMD(MMX, SSE) 명령어 현재 지원 안됨 ([Active issue](https://github.com/ish-app/ish/issues/57))

### blink shell
* 앱스토어에서 25000원
* **직접 빌드하면 공짜**
* 외장 모니터 관련 버그(?) 존재
